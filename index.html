Backup dữ liệu hệ thống
Chưa backup

Tải file sao lưu toàn bộ bảng dữ liệu BHLD ở định dạng SQL để lưu trữ hoặc khôi phục khi cần.

  • Cần đăng nhập để thực hiện backup.
  • File tải về chứa lệnh CREATE TABLE, INSERT dữ liệu và TRIGGER.
  • Nên lưu bản backup định kỳ trước khi thao tác dữ liệu lớn.
